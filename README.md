# dormProject

## 简介

宿舍管理系统

## 项目背景

GXA 第三阶段项目课业

## 功能介绍

### 功能

- 1.发布公告以及公告信息的增删改查
- 2.学生信息的增删改查
- 3.宿舍信息的增删改查
- 4.管理员信息的增删改查
- 5.宿舍物品增删改查
- 6.个别模块关联查询

### 技术

- 1.数据库选用 SQLServer
- 2.界面使用了 boostrap 进行美化
- 3.后端采用 node js express 框架

run with node and the use of database of mysql
