# DormProject 接口文档

## 公告

### 1. 获取公告

| 获取公告                                                    | 内容                                                         |
| :---------------------------------------------------------- | ------------------------------------------------------------ |
| 地址                                                        | /news/getnews                                                |
| 参数                                                        | 无                                                           |
| 请求类型                                                    | get                                                          |
| 返回数据<br/>Code 200 成功 data [ ] 数组<br/>数组内容:<br/> | id:1<br/>userId:1(用户id)<br/>userName:"1 号管理员"(发布人的名字)<br/>title:"公告1修"<br/>content:"公告1修"<br/>uploadTime:"2020-9-01 00:00:00"(创建时间)<br/>time:"2020-08-31T16:00:00.000Z"<br/> |
